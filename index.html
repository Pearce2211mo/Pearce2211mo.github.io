Senior Project: Purchase Request System
2020-2021 (c#/HTML/CSS)

The goal of this project is to create a new and improved Purchase Request System for the Missouri Department of Revenue.This new system will manage the creation of users and the users' purchasess. These purchases will be put through an approval process within the system. Each purchase request may have multiple items and may be purchased for multiple deparments requiring multiple accounting lines. There are four different types of users: admin, approver, requester, buyer. There are nine unique web pages.

2D Side-Scroller Spelling Game for Children 2019 (Java)

This is an interactive children's game built in Java. It contains cells and rows of colored tiles that flash images to imulate an old 2D side-scroller. The first half of the game involves moving boats and islands. The player is able to move their boat up and down to avoid collsions, as well as shoot cannon balls into oncoming enemy ships. The secound half of the game begins when users dock at an island. They are able to take control of a pirate and search for teasure. However they are only able to optain the treasure/ points by answering spelling questions correctly. This is a fun interactive game with a homescreen, leaderboard, and many fun secrets.

Experimental Self Check-out 2019 (Java)

This project was the start of creating a look up system for a self check-out. This project contains all methods you would need to be the foundation of a UI system. Instead of scanning items, you instead input them by a code as if a computer would do after a user scanned an item. Some functionalities include getting and setting codes and prices, adding discounts, and comparing products.
